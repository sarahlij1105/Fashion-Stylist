# Agentic Personal Stylist

An AI-powered fashion stylist application that helps users curate outfits based on their inventory or create new looks.

## Features

- **Vision Analysis**: Upload a photo to detect your current style and items.
- **Smart Recommendations**: Get outfit suggestions based on occasion, weather, and preferences.
- **Agentic Workflow**: Uses multiple AI agents for style analysis, procurement, and curation.

## Tech Stack

- React + Vite
- Tailwind CSS
- Google Gemini API
- Vercel Serverless Functions
